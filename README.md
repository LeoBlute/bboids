# bboids
### bboids or bevy_boids is an implementation of the boid/flocking algorithm in the bevy engine/ecosystem<br />

### the porpuse of the algorithm is to simulate "bird-like" behavior though 3 simple rules:<br />
1. separation<br />
1. alignment<br />
1. cohesion<br />
